### C++ Programs on Linux

- Compile: `$ g++ add.cpp`
- Run: `$ ./a.out`

### CUDA Programs on Linux

- Compile: `$ nvcc add_cuda.cu -o add_cuda`
- Run: `./add_cuda`
- Profile: `$ nvprof ./add_cuda`
